Backend & API Communication

This app's backend is powered by WordPress WooCommerce and the Dokan multivendor plugin. The frontend is developed using React Native. The backend communicates with the app through REST APIs provided by WooCommerce and Dokan.

  • WooCommerce REST API: Facilitates endpoints for retrieving product information, order details, and managing customer accounts.
  • Authentication: Uses JWT (JSON Web Token) or OAuth for secure communication.
  • Data Fetching: React Native app sends HTTP requests to these APIs using libraries like Axios or Fetch.
  • Real-time Updates: Webhooks from WooCommerce trigger notifications in the app for order updates or inventory changes.
  • Custom Endpoints: Additional endpoints can be created for specific app needs using WordPress plugins or custom PHP code.
